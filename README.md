## 🚀 Featured Project: AI Rigged

**[ai_riged](https://github.com/valeri78888/ai_riged)** - *Skeleton Detection & Visualization using MediaPipe*

A real-time skeletal visualization system that detects and displays all bones on camera using **MediaPipe**. This project showcases the power of pose detection and computer vision for interactive applications.

### What it does:
- ✨ Detects human skeletal structure in real-time
- 🎯 Visualizes bones and joints on live camera feed
- 📱 Powered by Google's MediaPipe framework
- 🎮 Perfect for fitness tracking, animation, and motion capture applications
